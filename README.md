# SHOPNOW-ECOMMERCE
## 🚀 Features
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
* **Dynamic Product Grid:** Products are rendered dynamically via JavaScript.
* **Category Filtering:** Sort products by Electronics, Clothing, Books, etc.
* **Real-time Search:** Filter products instantly using the search bar.
* **Functional Shopping Cart:** * Slide-out cart sidebar.
    * Real-item count badges.
    * Total price calculation.
* **Interactive UI:** Hover effects, sticky navigation, and smooth transitions.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Flexbox, Grid, and Custom Animations.
* **JavaScript (ES6):** DOM manipulation and state management.
